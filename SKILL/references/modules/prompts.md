# Prompts Plugin (v1.4.9+)

全局提示词标签库，**不绑定项目**。TitleBar 💡 按钮一键打开弹窗，按类目浏览
内置标签 + 用户自定义标签，点击追加到撰写区，编辑好后一键复制到剪贴板，
回到任意提示词输入框手动粘贴。

## 数据模型

- 全局 SQLite：`APPDATA/LumiCreate-Pro/prompts.sqlite`
- 表 `prompt_tags`：`id / category / name / content / description / is_builtin / sort_order / created_at`
  - `name` 中文显示名（"动漫" / "黄昏" / "倾斜镜头"）
  - `content` 真正复制的提示词文本（通常英文："anime style" / "golden hour"）
  - `is_builtin=1` 表示内置出厂集，**不能编辑 / 不能删除**（只能整体 reset）
  - `is_builtin=0` 表示用户自定义，可改可删

## API（前缀 `/api/prompts`）

| Method | Path                | Auth | Note |
|--------|---------------------|------|------|
| GET    | `/categories`        | ✗    | 去重 category 列表，内置在前 |
| GET    | `/list?category=&limit=` | ✗ | 标签列表，内置在前，自定义在后；`limit` 默认 1000 |
| POST   | `/tag`               | ✗    | 创建自定义；body: `{category, name, content, description}` |
| PUT    | `/tag/{id}`          | ✗    | 改自定义（内置返 400） |
| DELETE | `/tag/{id}`          | ✗    | 删自定义（内置返 400） |
| POST   | `/reset-builtins`    | ✗    | 清掉内置 + 重塞出厂集；不影响自定义 |

## 弹窗 UX

```
┌─ 💡 提示词插件 ──────────────────────────── 提示语 ──── ✕ ─┐
│ ┌── 类目（7） ──┐  ┌── 标签栏 ────────────────────────┐ │
│ │ 画风（active）│  │ 🔍 搜索…  [＋ 新增自定义]        │ │
│ │ 构图          │  │ ┌─── 新增表单（折叠）───────┐    │ │
│ │ 光照          │  │ └────────────────────────────┘    │ │
│ │ 情绪          │  │ ┌─ 标签网格（auto-fill 180px）─┐  │ │
│ │ 角色          │  │ │ [动漫]   [漫画]    [厚涂]    │  │ │
│ │ 画质          │  │ │ [水彩]   [吉卜力]  [皮克斯]  │  │ │
│ │ 负面词        │  │ │ [写实]   [国风]    [赛博朋克]│  │ │
│ │ ↺ 重置内置    │  │ │ [蒸汽朋克] [自定义A]✕  …    │  │ │
│ └───────────────┘  │ └────────────────────────────────┘ │
│                    └──────────────────────────────────────┘│
│ ┌─ 撰写区 — 84 字符 ───────────────  ✓ 已复制 84 字符 ─┐  │
│ │ anime style, golden hour, dynamic pose             │  │
│ └─────────────────────────────────────────────────────┘  │
│ 🗑 清空  · 分隔符 ⊙ 逗号 ○ 空格 ○ 换行   📋 复制到剪贴板 │
└─────────────────────────────────────────────────────────────┘
```

- 点击标签 → 追加到撰写区（按当前分隔符）
- 撰写区可自由键入 / 编辑
- 「📋 复制到剪贴板」 → 用 `navigator.clipboard.writeText`，老版浏览器走
  `execCommand('copy')` 兜底
- 分隔符在逗号 / 空格 / 换行间切换（默认逗号 + 空格）
- 自定义标签左侧有黄边，hover 出现右上角 ✕ 删除按钮
- 「↺ 重置内置」清掉内置 + 重塞出厂集（用户自定义保留）

## 实操推荐

- 第一次打开自动 seed 出厂集，开箱即用
- 经常用的画风组合可以自己写一个自定义（如 "我的厚涂"，content 拼 N 个内置词），加到「自定义」类目
- 写负面词时把分隔符切到逗号，整体复制到 ComfyUI 的 negative_prompt 字段
- 写镜头描述时把分隔符切到空格，更贴 SD 自然语言风格
